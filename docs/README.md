# Documentación

Este directorio reúne la documentación operativa y técnica de ToppFinance.

## Índice

- `architecture.md`: mapa de la arquitectura del monorepo y responsabilidades por capa.
- `development.md`: flujo de desarrollo local y onboarding técnico.
- `commands.md`: inventario de scripts y secuencias recomendadas.
- `deployment.md`: checklist y pasos mínimos de despliegue.
- `environment.md`: variables de entorno y reglas de mantenimiento.
- `runbook.md`: checklists operativos e incidencias frecuentes.
- `backups.md`: estrategia y uso de copias de seguridad.
- `csv-import.md`: notas sobre importación de datos.
- `ai-privacy.md`: consideraciones de privacidad e integraciones AI.

## Orden de lectura recomendado

1. `../README.md`
2. `development.md`
3. `environment.md`
4. `commands.md`
5. `architecture.md`
6. Resto de documentos según la tarea
